MCP server.json Validator & Skeleton Generator

Validate an MCP server.json against the 2025-12-11 generic schema and the official MCP Registry publishing rules: reverse-DNS namespace, the _meta 4 KB publisher cap, allowlisted registry base URLs, and the MCPB fileSha256 requirement: then generate a compliant skeleton for an npm, PyPI, OCI, MCPB, or remote server. Everything runs locally; nothing is published or transmitted.

⚠ The MCP Registry and server.json schema are in preview and may change or reset. Rules embedded here track schema 2025-12-11. Re-verify against the live registry docs before publishing.
